# HR Data Analysis with Python

##  Project Overview

This project analyzes HR employee data using Python to uncover workforce trends and generate actionable business insights. It covers data cleaning, exploratory data analysis (EDA), and visualization to understand employee distribution, salary patterns, performance, experience, hiring trends, and attrition. The findings support data-driven HR decision-making and workforce planning.

##  Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

##  Project Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Data Exploration
4. Exploratory Data Analysis (EDA)
5. Data Visualization
6. Business Insights & Recommendations

##  Business Questions Answered

* What is the distribution of employees across departments?
* Which departments offer the highest average salaries?
* Which job titles receive the highest salaries?
* How does salary vary with years of experience?
* How does employee performance differ across departments?
* What are the hiring trends over the years?
* Is there a salary difference between remote and on-site employees?
* Which departments have the highest attrition?
* Who are the highest-paid employees in each department?
